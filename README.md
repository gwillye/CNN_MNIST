# CNN-MNIST — Convolutional Neural Networks on MNIST

> Classic CNN architectures (**LeNet** & **AlexNet**) implemented and trained on the MNIST handwritten-digit dataset.

## 🌎 Context
Deep-learning study project (UFMS — Information Systems), implementing and comparing foundational CNN architectures from scratch.

## 📓 What's here
- `CNN_para_MNIST_LeNet.ipynb` — LeNet-style architecture.
- `CNN_para_MNIST_AlexNet.ipynb` — AlexNet-style architecture adapted to MNIST.

## 🛠️ Tech
Python · TensorFlow / Keras · CNN (LeNet, AlexNet) · MNIST
